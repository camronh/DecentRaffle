## Instructions

```
cd frontend
npm install
npm run serve
```

### To deploy on a local blockchain

> Be sure to create a `.env` file in the hardhat dir that matches the `.env.example` template

```
cd hardhat
npm install
npm run local-node
npm run deploy-local
```
